\doxysubsection{Detailed Description}
Sequential Impulse contact solver for unconstrained rigid bodies. 

Resolves normal and friction contact constraints using a Projected Gauss-\/\+Seidel loop with accumulated impulse clamping. Contact impulses are computed in world coordinates and applied to linear velocity (m/\+s) and angular velocity (rad/\+s).

\begin{DoxyNote}{Note}
This solver is intended for a single-\/threaded update loop. 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Inputs must be physically consistent (mass in kg, inertia in kg\texorpdfstring{$\ast$}{*}m\texorpdfstring{$^\wedge$}{\string^}2, time step in seconds) to avoid instability. 
\end{DoxyWarning}
