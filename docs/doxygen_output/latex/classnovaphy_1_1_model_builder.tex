\doxysubsection{Detailed Description}
Builder utility for constructing immutable free-\/body scene models. 

Collects rigid bodies, their initial transforms, and collision shapes, then emits a {\ttfamily \doxylink{structnovaphy_1_1_model}{Model}} snapshot suitable for simulation.
