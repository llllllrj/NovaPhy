\doxysubsection{Detailed Description}
Per-\/body simulation state buffers for free rigid-\/body stepping. 

Stores pose, velocity, and accumulated wrench terms for each body. Linear quantities are in world coordinates (m, m/\+s, N), angular quantities are in world coordinates (rad/\+s, N\texorpdfstring{$\ast$}{*}m).
