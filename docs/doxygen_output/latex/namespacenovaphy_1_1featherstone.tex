\doxysubsection{Detailed Description}
Featherstone-\/family rigid-\/body dynamics algorithms. 

This namespace provides forward kinematics, inverse dynamics (RNEA), mass-\/matrix assembly (CRBA), and forward dynamics for a single tree-\/structured articulation.

Spatial-\/vector convention is \+[angular; linear]\+, and computations follow standard rigid-\/body dynamics in SI units (m, kg, s, N, rad).
