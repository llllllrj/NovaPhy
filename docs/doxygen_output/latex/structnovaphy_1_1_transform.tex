\doxysubsection{Detailed Description}
Rigid transform represented by translation and quaternion orientation. 

{\ttfamily \doxylink{structnovaphy_1_1_transform_a074b6b85886aacd125ea8db47cc25f80}{position}} is expressed in world coordinates (m). {\ttfamily \doxylink{structnovaphy_1_1_transform_a1a88b660d881eab306cb61d9a3218707}{rotation}} stores attitude as a unit quaternion for stable composition and interpolation.
