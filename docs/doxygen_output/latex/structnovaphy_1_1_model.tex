\doxysubsection{Detailed Description}
Immutable free-\/body scene model. 

Describes rigid bodies, their initial world transforms, and collision shapes. Instances are typically created with {\ttfamily \doxylink{classnovaphy_1_1_model_builder}{Model\+Builder}}.
