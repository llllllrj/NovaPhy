\doxysubsection{Detailed Description}
High-\/level free-\/rigid-\/body simulation container. 

Owns model topology, mutable simulation state, broadphase collision detector, and contact solver. {\ttfamily \doxylink{classnovaphy_1_1_world_abc197156cc4e9344e6a90f2df07c577d}{step()}} advances one fixed-\/size time step in SI units using gravity, collision detection, and impulse-\/based resolution.
