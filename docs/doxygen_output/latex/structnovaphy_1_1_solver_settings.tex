\doxysubsection{Detailed Description}
Runtime tuning parameters for the free-\/body contact solver. 

These parameters control convergence and stability of the Projected Gauss-\/\+Seidel (Sequential Impulse) contact solve in SI units. For dense stacks, increasing iteration count and using moderate Baumgarte stabilization typically reduces jitter and drift.
