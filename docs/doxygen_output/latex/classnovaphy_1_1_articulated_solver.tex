\doxysubsection{Detailed Description}
Time-\/stepping solver for articulated-\/body dynamics. 

Wraps Featherstone forward dynamics and advances generalized state using semi-\/implicit integration. Quaternion state components are normalized after integration for free/\+ball joints.
