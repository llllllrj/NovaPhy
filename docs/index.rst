PhysicsEngine Documentation
===========================

Welcome to the unified C++ and Python API documentation for PhysicsEngine.

Installation
------------

1. Install documentation dependencies:

   .. code-block:: bash

      pip install -r docs/requirements.txt

2. Build/install the pybind11 Python module (for autodoc import).
3. Run ``./build_docs.sh`` to generate API docs and HTML output.
